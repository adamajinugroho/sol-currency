This currency directives use standard from local country using angular i18n
for example in Indonesia use EBI, Rp1.000,00 and for negative -Rp1.000,00
in France/German etc, 1 000,00 € and for negative -1 000,00 €
in US etc, $1,000.00 and for negative -$1,000.00
Also this directives can use ISO 4217 or (3-letter alphabet) to distinguish e.g. USD and SGD

The comparison from angular built-in currency pipe is this directives internally use Service for Locale Config
so that developer doesnt need to include the service in every single component
it is also reactive to locale config change
